**Goal:** Implement billing and tiered features

- [ ] **Stripe Account Setup** - Create and configure Stripe account
- [ ] **Subscription Plans** - Set up products in Stripe (Free, Starter $9, Pro $49)
- [ ] **Payment Flow** - Build checkout and subscription management pages
- [ ] **Usage Tracking System** - Count conversations per widget/account

  - [ ] Unlimited conversations

- [ ] **Conversation Analytics Dashboard** - Charts for volume, topics, response times
